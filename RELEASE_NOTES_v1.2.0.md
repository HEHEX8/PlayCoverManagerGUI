# PlayCover Manager v1.2.0 リリースノート

**リリース日**: 2025年11月13日

## 🎉 主な変更点

### 🎨 UI/UX の大幅改善

#### 1. 統一モーダルシステム ✨
すべてのアラート・プロンプトを完全に統一し、一貫したユーザー体験を実現：

- **画面中央表示**: すべてのモーダルが正しく画面中央に表示されるようになりました
- **ガラスエフェクト**: macOS 26 Liquid Glass デザインに完全対応
- **キーボード操作**: Return（実行）/ Escape（キャンセル）で素早く操作可能

**対象のモーダル**:
- エラーメッセージ
- ディスクイメージ作成確認
- 言語変更の再起動プロンプト
- データハンドリング戦略選択
- 設定リセット確認
- キャッシュクリア確認
- ショートカット削除結果

#### 2. アイコングリッドレイアウト最適化 📐
小さなウィンドウでも快適に使用できるよう改善：

- **動的カラム数**: アプリ数に応じて1-10カラムを自動調整
- **最大アイコンサイズ**: ウィンドウサイズに応じてアイコンを最大化
- **完全な中央揃え**: 視覚的に完璧な中央配置を実現
- **全アプリ表示**: どのウィンドウサイズでもすべてのアプリが表示されます

**Before**: 9番目のアプリが画面外に隠れる、右寄りの表示  
**After**: すべてのアプリが表示、完璧な中央揃え

#### 3. UIスペーシング削減 📏
不要なスペースを削減し、情報密度を向上：

- **設定画面**: 過剰な行間隔を削減
- **IPAインストール**: 1アプリでもスクロール不要に
- **全体的な改善**: より多くの情報が一目で見えるように

#### 4. ツールバーボタン改善 🔴
視認性と操作性を向上：

- **全イジェクトボタン**: 赤色⏏︎アイコン（22pt）で強調
- **ヘルプボタン**: サイズ拡大（22pt）

---

## 🐛 修正されたバグ

### クリティカルな修正

1. **モーダル表示位置の問題** ✅
   - ScrollView内に表示されて正しい位置にならない問題を完全解決
   - すべてのモーダルが画面中央に正しく表示されるようになりました

2. **アイコングリッドの中央揃え** ✅
   - 右寄りだったアイコン配置を完璧な中央揃えに修正
   - ツールバーとグリッドのパディングを統一（32pt）

3. **9番目のアプリ非表示問題** ✅
   - 最小サイズ制約を削除し、常にすべてのアプリを表示
   - アイコンが縮小されても全て表示されます

4. **ビルドエラーの完全解決** ✅
   - 古いアラートシステムの参照をすべて削除
   - StandardAlert, KeyboardNavigableAlert, AlertButton の全参照を削除
   - GeneralSettingsView の不要な Binding パラメータを削除
